If creating visual artifacts (slides, mocks, throwaway prototypes, etc), copy assets out and create static HTML files for the user to view. If working on production code, you can copy assets and read the rules here to become an expert in designing with this brand.
If the user invokes this skill without any other guidance, ask them what they want to build or design, ask some questions, and act as an expert designer who outputs HTML artifacts _or_ production code, depending on the need.

Key notes for this brand:
- Editorial, cinematic, warm, dark-dominant — NOT a SaaS or OTA look
- Cormorant Garamond (serif display, weight 300, often italic), Manrope (body), League Spartan (eyebrows ALL CAPS tracked)
- Palette: near-black `#0f1116`/`#0a0a0a`, warm cream `#f6f2e9`, wine-red `#8f2f2f`, champagne gold `#d3a65a`
- No emoji, no icon libraries (inline SVG only), no gradient fills except overlays/protection fades
- "We Care." is the brand leitmotif — the care IS the luxury
- Use `colors_and_type.css` as the single source of truth for tokens
